📐 Model Formulas

Front irradiance (equator‑facing, Liu‑Jordan isotropic):

beam_tilted = DNI × max(0, cos I)
sky_vf = (1 + cos β)/2
gnd_vf = (1 − cos β)/2
ghi_diffuse = GHI × kd
front = beam_tilted + ghi_diffuse×sky_vf + GHI×albedo×gnd_vf

Rear irradiance (BTP‑2, Yusufoglu et al. 2014, Eq. 9):

rear_vf = (1 + cos β)/2
E_rear = α×DHI×rear_vf + α×(GHI−DHI)×(rear_vf − F_V)
rear_effective = E_rear × bifaciality

Shadow View Factor (FV):

γ_profile = atan(tan(sun_elev) / cos(sun_az − panel_az))
shadow_lower = h / tan(γ_profile)
shadow_upper = (h + W·sin β) / tan(γ_profile)
F_V = rear_vf − ∫[shadow] cos θ₁·cos θ₂ / (2·r) dx

FV accounts for the module's own shadow blocking direct ground reflection. Solar azimuth is computed analytically from site coordinates.

Total effective = front + rear (≥0). No temperature correction applied.
